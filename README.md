# 🌱 SpringBoot-Learning

> A hands-on journey to master **Spring Boot REST APIs** — from zero to production-ready.

---

## 📌 About

This repository is my personal learning journal for Spring Boot. I started from absolute scratch and am progressively building real REST APIs, picking up backend concepts one commit at a time.

By the end of this journey, I aim to be confident in building, securing, and deploying production-grade Spring Boot applications.

---

## 🛠️ Tech Stack

- **Java 17+**
- **Spring Boot 3.x**
- **Maven**
- **REST API / Spring Web**
- **Spring Data JPA**
- **MySQL / H2**
- **Spring Security + JWT** *(planned)*

---

## 🎯 What I'm Learning

- REST API design with Spring Boot
- Dependency Injection & Spring IoC Container
- CRUD operations with JPA & Hibernate
- Database integration (MySQL)
- Input validation & global exception handling
- API security with Spring Security & JWT
- Deploying Spring Boot apps to the cloud

## 🚀 How to Run Any Project

```bash
# Clone the repository
git clone https://github.com/sumiran3031/SpringBoot-Learning.git

# Navigate into any project folder
cd SpringBoot-Learning/<folder-name>

# Run the app
./mvnw spring-boot:run
```

Then open: `http://localhost:8080`
